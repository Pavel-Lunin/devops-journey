# Learning Log

Еженедельные заметки о прогрессе на пути к Middle DevOps Engineer.

## Формат

Один файл на неделю: `YYYY-week-NN.md`. Каждая запись содержит секции:

- **What I did** — что сделано за неделю
- **What I learned** — ключевые insights
- **Blockers / questions** — что мешало, что неясно
- **Plan for next week** — план на следующую неделю
- **Reflections** — свободные размышления, корректировки курса

См. `2026-week-21.md` как шаблон-пример.

## Зачем

Публичный учебный дневник служит трём целям одновременно:

1. **Honest accountability** — невозможно "пропустить неделю" незаметно.
2. **Pattern recognition** — через 3 месяца видно, где realy buksuet и куда сместить фокус.
3. **Material for interviews** — конкретные истории прогресса полезны при ответах на behavioural-вопросы.
