# System Design Questions

Подборка system design вопросов из DevOps-интервью и мои решения.

Каждая задача — отдельный markdown-файл со структурой:

- **Problem statement** — что спросили, какие constraints
- **Clarifying questions** — что я уточнил у интервьюера (или у себя)
- **High-level design** — диаграмма + краткое описание
- **Deep dive** — узкие места и trade-offs
- **What I'd do differently** — рефлексия после интервью
