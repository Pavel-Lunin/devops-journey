# Interview Preparation

Материалы для подготовки к DevOps-интервью на РФ-рынке.

## Structure

- `system-design/` — типичные system design questions и мои решения
- `star-stories/` — STAR-методология stories из моего опыта (Сбер improvements, mobile-DevOps transition)
- (будет добавляться по мере прохождения интервью)

## Target companies

- **Tier 1:** Yandex, VK Tech, Tinkoff, Ozon, Wildberries, Сбер internal
- **Tier 2:** Альфа, Райффайзен, Авито, Kaspersky, Точка
- **Tier 3:** Selectel, Cloud.ru, Luxoft, EPAM

## Заметки по подготовке

Подход — итеративный: после каждого реального интервью добавляю заметку о том, какие вопросы встретились, где провалился, какие STAR-stories сработали, а какие нет.
