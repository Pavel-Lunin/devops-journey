# Docker Course Labs

Практические упражнения из отдельного учебного проекта по Docker, перенесённые в project 03 как фундамент для будущего React Native Android builder.

| Lab | Тема | Артефакт |
|---|---|---|
| `python-app` | Сборка image и интерактивный container | Python CLI-календарь |
| `node-app` | Bind mount и запись данных на host | `my-file.txt` |
| `nginx` | Read-only bind mount и port mapping | Статическая HTML-страница |
| `docker-compose-app` | Compose network и service discovery | Python client + MongoDB |

Команды запуска и известные ограничения зафиксированы в [README проекта](../README.md).
